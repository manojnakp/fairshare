## Overview

Share bills and keep track of the group expenses in a fair and easy way.

## Installation

### Using `go install`

Pre-Requisites:
- [Go]

```sh
go install github.com/manojnakp/fairshare@latest
```

### Build from source

Pre-Requisites:
- [Go]
- [Git]

```sh
git clone https://github.com/manojnakp/fairshare
cd fairshare
go build . -o fairshare
```
